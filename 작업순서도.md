# DustMate 앱 초안 제작 작업순서도

## 📋 개요
PRD 기반으로 HTML/CSS/JavaScript를 활용하여 DustMate 웹앱 인터랙티브 초안을 제작합니다.
백엔드 연동 없이 **목업 데이터**를 사용하는 프론트엔드 프로토타입입니다.

---

## 🗂️ 최종 파일 구조

```
DustMate/
├── index.html          ← 메인 진입점 (SPA 방식, 화면 전환 JS로 처리)
├── style.css           ← 전역 스타일 + 디자인 시스템
├── app.js              ← 앱 전체 로직 (라우팅, 데이터, 상태 관리)
├── DustMate_PRD.md
└── 작업순서도.md
```

---

## 🎨 디자인 시스템 (style.css 우선 정의)

| 항목 | 내용 |
|------|------|
| **테마** | 다크모드 기반, 미세먼지 농도에 따른 동적 컬러 |
| **주조색** | 맑음 `#4FC3F7` / 보통 `#A5D6A7` / 나쁨 `#FFB74D` / 매우나쁨 `#EF5350` |
| **배경색** | `#0D1117` (딥 다크) |
| **카드 배경** | `#161B22` + glassmorphism 효과 |
| **폰트** | Noto Sans KR (Google Fonts) |
| **애니메이션** | 화면 전환 fade/slide, 카드 hover lift |

## 🚦 작업 단계

### STEP 1 — 프로젝트 기반 세팅

---

### STEP 2 — 화면 1: 로그인 / 회원가입

**구현 내용:**

- [ ] 로그인 성공 시 홈 화면으로 이동 (JS)

### STEP 3 — 화면 2: 홈 화면

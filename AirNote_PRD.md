# Dust Mate PRD (초안)

## 1. 한 줄 소개

미세먼지 위험 정보를 친구·가족과 공유하고 건강 상태를 기록하는 소셜 건강 다이어리 웹앱

---

## 2. 목표

### 목표 1
사용자가 실시간 미세먼지 경보 정보를 쉽게 확인할 수 있도록 한다.

### 목표 2
친구·가족과 대기질 위험 정보를 공유하여 건강 관리 습관을 형성한다.

### 목표 3
미세먼지와 개인 건강 상태의 관계를 기록하고 추적할 수 있도록 한다.

---

## 3. 타겟 사용자

### 주요 타겟

- 20~40대 건강 관심 사용자
- 어린 자녀를 둔 부모
- 부모님 건강을 챙기는 성인 자녀
- 야외활동이 많은 직장인 및 운동 동호인

### 부가 타겟

- 시니어 사용자
- 호흡기 질환 관리가 필요한 사용자

---

## 4. 핵심 기능

| 기능 | 구분 | 설명 |
|--------|--------|--------|
| 실시간 미세먼지 경보 조회 | 필수 | 에어코리아 API를 활용한 지역별 경보 현황 조회 |
| 건강 다이어리 작성 | 필수 | 컨디션, 외출 여부, 증상 기록 |
| 친구·가족 그룹 관리 | 필수 | 친구 초대 및 그룹 생성 |
| 위험 알림 공유 | 필수 | 경보 발생 시 친구·가족에게 알림 전송 |
| 건강 기록 통계 조회 | 선택 | 기간별 건강 상태 및 기록 분석 |

---

## 5. 사용자 흐름

1. 사용자가 회원가입 또는 로그인한다.
2. 현재 지역을 설정한다.
3. 메인 화면에서 미세먼지 경보 상태를 확인한다.
4. 건강 다이어리를 작성한다.
5. 친구 또는 가족을 그룹에 초대한다.
6. 경보 발생 시 그룹원에게 알림이 전달된다.
7. 사용자는 과거 건강 기록 및 통계를 확인한다.

---

## 6. 화면 목록

### 1. 로그인 / 회원가입
- 카카오 로그인
- 이메일 로그인
- 회원가입

### 2. 홈 화면
- 현재 지역 표시
- 미세먼지 상태 표시
- 경보 현황 표시
- 오늘의 건강 기록 요약

### 3. 건강 다이어리 작성
- 컨디션 입력
- 증상 체크
- 외출 여부 기록
- 메모 작성

### 4. 친구·가족 그룹
- 친구 추가
- 그룹 생성
- 그룹 멤버 목록

### 5. 기록 히스토리
- 일별 기록 조회
- 캘린더 보기
- 통계 보기

# 추가로 확인할 질문 10개

1. 회원가입은 카카오 로그인만 사용할 것인가?
2. 위치 정보는 GPS를 사용할 것인가, 직접 선택할 것인가?
3. 친구 추가 방식은 초대 링크 방식인가?
4. 그룹 최대 인원 수는 몇 명으로 제한할 것인가?
5. 푸시 알림 기능을 MVP에 포함할 것인가?
6. 건강 기록은 하루 1회만 작성 가능한가?
7. 기록 공개 범위는 본인만 보기인가, 그룹 공유인가?
8. 미세먼지 외에 초미세먼지 정보도 함께 제공할 것인가?
9. 통계 기능은 MVP에 포함할 것인가, 추후 개발할 것인가?
10. 모바일 웹앱 이후 네이티브 앱으로 확장할 계획이 있는가?
